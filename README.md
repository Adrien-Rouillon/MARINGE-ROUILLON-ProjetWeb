# MARINGE-ROUILLON-ProjetWeb

Projet Web 2026 de Alexis Maringe et Adrien Rouillon
Le Projet est de faire un site vitrine du departement d'informatique.
Il contient:
  - Une page d'acceuil avec un carroussel des chiffres clés de l'EFREI et les actualités.
  - Une page sur l'equipe de plusieurs département.
      - fonctionne a partir d'un JSON: si vous rajoutez une personne dans le fichier JSON, il est automatiquement rajouté dans la liste sur le site web
  - Une page vie étudiante avec des informations sur les instalations de l'EFREI, des associations et des projet.
  - Une page contact avec un formulaire qui peut etre remplie. Une fois le commentaire envoyé, celui-ci est stocké avec le nom et l'adresse e-mail de l'expéditeur dans le stockage local (Local Storage) de votre navigateur.
Pour y accéder et consulter les données, suivez ces étapes :
-Faites un clic droit n'importe où sur la page.
-Sélectionnez Inspecter.
-Allez dans l'onglet Application (en haut de la console).
-Dans le menu de gauche, déroulez la section Local Storage.
-Sélectionnez l'URL de votre site pour afficher le contenu enregistré.

  - Une page à-propos avec des information concernant les deux etudiant qui on travaillé sur ce projet. Vous pourrez y accedée en cliquant sur le lien présent dans le footer " A propos de nous "

Les fichier professeur.json a été remplie grace à l'IA.
